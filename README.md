# 🤖 RestoBot

**Assistant WhatsApp intelligent pour restaurants au Mali**

Pendant que vous cuisinez, RestoBot prend les commandes.

---

## ✨ Fonctionnalités

- 🧠 **IA intelligente** - Comprend "2 poulets sans piment" naturellement
- 💳 **Paiement mobile** - Wave, Orange Money, Moov Money
- 📊 **Dashboard temps réel** - Commandes, revenus, statistiques
- 📸 **Photos du menu** - Jusqu'à 5 photos envoyées automatiquement
- 🏪 **Multi-restaurants** - Gérez tous vos établissements
- 📱 **WhatsApp natif** - Zéro application à télécharger

---

## 🚀 Démarrage rapide

```bash
# Cloner le repo
git clone https://github.com/TON_USERNAME/restobot.git
cd restobot

# Backend
cd backend
npm install
cp .env.example .env  # Configurer vos clés
node src/server.js

# Frontend (nouveau terminal)
cd frontend
npm install
npm run dev
